Propriedades de Dimensionamento e Espaçamento

largura e altura

basico:

width: 123px;
height: 321px;

auto, initial e inherit

auto - ocupa toda a largura ou altura do conteudo em si
initial - valor padrao

div pai
div filha
inherit - valor pai

altura e largura máxima e mínima

min-width: 200px

min-height: 50px

max-witdh: 100px

max-height: 30px

margin

Aplicando a mesma margem a todos os lados
.exemplo1 {
margin: 10px;
}

Aplicando margens diferentes aos lados
.exemplo2 {
margin: 5px 10px 15px 20px;
}
cima: 5px, direita: 10px, baixo: 15px, esquerda: 20px

Margens iguais no topo/fundo e direita/esquerda

.exemplo3 {
margin: 8px 12px;
}
top/bottom: 8px, right/left: 12px

padding

mesma lógica do margin, soq invés de espaçamento entre elementos
é o espaçamento entre o elemento e seu conteúdo de dentro
mas tem um porém, esse espaçamento é adicionado, nao definido, então o tamanho da div também aumenta
então utilizamos o Box Sizing

box-sizing

box-sizing: border-box;

assim a div fica com o tamanho original incluindo os paddings
