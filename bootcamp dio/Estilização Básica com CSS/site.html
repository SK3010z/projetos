Fundos

Propriedades de Fundo:

  1. background-color: Define a cor de fundo do elemento.
  2. background-image: Especifica uma imagem para ser usada como fundo.
  3. background-repeat: Controla como a imagem de fundo é repetida.
  4. background-position: Define a posição inicial da imagem de fundo.
  5. background-size: Especifica o tamanho da imagem de fundo.
  6. background-attachment: Controla se a imagem de fundo rola com o restante da página ou permanece fixa.

Propriedades Adicionais:

  1. background-origin: Define a origem do posicionamento da imagem de fundo.
  2. background-clip: Define a área de exibição do fundo.
  3. background-blend-mode: Determina como o fundo se mescla com o conteúdo do elemento.
  4. background: Combina várias propriedades de fundo em uma única regra.

Essas informações fornecem uma base abrangente para trabalhar com fundos em HTML e CSS, oferecendo controle detalhado sobre a aparência visual dos elementos em uma página web.



Bordas

1. Propriedades Básicas:

  • border-width: Define a largura da borda.
  • border-style: Define o estilo da borda (solid, dashed, dotted, etc.).
  • border-color: Define a cor da borda.
  • border: Combina todas as propriedades da borda em uma única regra.

2. Propriedades Individuais:

  • border-top: Propriedades específicas para a borda superior.
  • border-right: Propriedades específicas para a borda direita.
  • border-bottom: Propriedades específicas para a borda inferior.
  • border-left: Propriedades específicas para a borda esquerda.

3. Propriedades Avançadas:

  • border-radius: Define o raio dos cantos da borda (para arredondar).
  • border-image: Permite o uso de uma imagem como borda.
  • border-collapse: Controla a modelagem de bordas em tabelas.
  • border-spacing: Define a distância entre as células de uma tabela.

4. Estilos de Borda:

  • solid: Borda sólida.
  • dashed: Borda tracejada.
  • dotted: Borda pontilhada.
  • double: Borda dupla.
  • groove: Borda entalhada.
  • ridge: Borda em relevo.
  • inset: Borda em relevo interno.
  • outset: Borda em relevo externo.


Fontes

1. Propriedades Básicas:

  • font-family: Define a família da fonte (serif, sans-serif, monospace, etc.).
  • font-size: Define o tamanho da fonte.
  • font-weight: Define a espessura da fonte (normal, bold, bolder, lighter, etc.).
  • font-style: Define o estilo da fonte (normal, italic, oblique).
  • line-height: Define a altura da linha. (definida na pagina como 25px)
  • color: Define a cor do texto.

2. Propriedades Avançadas:

  • @font-face: Permite o uso de fontes personalizadas.
  • font-variant: Controla a variante da fonte (normal, small-caps).
  • font-stretch: Controla a largura da fonte. ( condensed, expanded)

Exemplo:

```css
body {
font-family: "Arial", sans-serif;
font-size: 16px;
font-weight: bold;
font-style: italic;
line-height: 1.5;
color: #333;
}
```

Exemplo com @font-face:

```css
@font-face {
font-family: "MinhaFontePersonalizada";
src: url('caminho/para/minha-fonte.woff') format('woff');
font-weight: normal;
font-style: normal;
}

body {
font-family: "MinhaFontePersonalizada", sans-serif;
}
```

Exemplo com @import:

```css
@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap');

body {
font-family: 'Roboto', sans-serif;
}
```


Textos

1. Text Transform:

text-transform: Transforma o texto para:

  • uppercase: Transforma todo o texto em maiúsculas.
  • lowercase: Transforma todo o texto em minúsculas.
  • capitalize: Transforma a primeira letra de cada palavra em maiúscula.

2. Text Align:

text-align: Alinha o texto para:

  • left: Alinha o texto à esquerda.
  • right: Alinha o texto à direita.
  • center: Centraliza o texto.

3. Text Decoration:

text-decoration: Adiciona decoração ao texto:

  • underline: Adiciona sublinhado ao texto.
  • overline: Adiciona uma linha acima do texto.
  • line-through: Adiciona uma linha no meio do texto.
  • none: Remove qualquer decoração do texto.

4. Text Indent:

text-indent: Define o recuo da primeira linha do texto, exemplo: text-indent: 20px;

5. Letter Spacing e Word Spacing:

letter-spacing: Define o espaçamento entre caracteres, exemplo: letter-spacing: 2px;

word-spacing: Define o espaçamento entre palavras, exemplo: word-spacing: 10px;

6. White Space:

white-space: Controla o tratamento do espaço em branco:

  • normal: Trata o espaço em branco de forma padrão.
  • nowrap: Não permite quebra de linha.
  • pre: Mantém espaçamento e quebras de linha, nao quebra textos sozinho.
  • pre-line: Mantém quebras de linha, mas ignora espaços extras.
  • pre-wrap: Mantém espaçamento e quebras de linha, quebra textos sozinho.
  • break-spaces: Mantém espaçamento mas quebra de tudo, até espaços

7. Word Wrap:

word-wrap: Controla a quebra de palavras em uma linha:

  • normal: Permite que as palavras quebrem no final da linha.
  • break-word: Força as palavras a quebrarem caso não caibam na linha.

8. Word Break:

word-break: Define como as palavras podem ser quebradas:

  • normal: Quebra as palavras de acordo com as regras padrão.
  • break-all: Força a quebra de palavras, mesmo no meio delas.
  • keep-all: Tenta manter as palavras juntas quando possível.

9. Writing Mode:

writing-mode: Define o modo de escrita do texto:

horizontal-tb:

Modo padrão, da esquerda para a direita.

vertical-rl:

Modo vertical, da direita para a esquerda.

vertical-lr:

Modo vertical, da esquerda para a direita.

10. Text Overflow:

text-overflow: Define como o conteúdo é tratado quando não cabe em uma caixa:

  • clip: Corta o texto que ultrapassa a caixa.
  • ellipsis: Adiciona reticências (...) ao final do texto que não cabe.


Sombras

1. Box Shadow:

box-shadow: Adiciona uma sombra a um elemento:

  • h-shadow: Deslocamento horizontal da sombra.
  • v-shadow: Deslocamento vertical da sombra.
  • blur: Espalha a sombra, tornando-a mais difusa.
  • spread: Expande ou contrai a sombra.
  • color: Cor da sombra.
  • inset: Cria uma sombra interna em vez de externa.

2. Text Shadow:

text-shadow: Adiciona uma sombra ao texto:

  • h-shadow: Deslocamento horizontal da sombra.
  • v-shadow: Deslocamento vertical da sombra.
  • blur: Espalha a sombra, tornando-a mais difusa.
  • color: Cor da sombra.

3. Multiple Shadows:

Permite aplicar várias sombras em um único elemento, separando-as por vírgulas.

4. Box Reflect:

box-reflect: Cria um reflexo do conteúdo em um elemento.

5. Filter Drop Shadow:

filter: com drop-shadow() adiciona uma sombra com mais flexibilidade em relação ao posicionamento e tamanho.


Overflow

1. Valor da Propriedade:

overflow: Define como o conteúdo deve se comportar quando excede as dimensões do contêiner.

2. Valores Principais:

  • visible: O conteúdo é exibido fora do contêiner.
  • hidden: O conteúdo é cortado e não é visível.
  • scroll: Adiciona barras de rolagem, mesmo que não sejam necessárias.
  • auto: Adiciona barras de rolagem apenas quando o conteúdo excede as dimensões do contêiner.
